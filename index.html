Business Information Systems FAQs

Q.1 Answer the followings.

A. Answer / Define the following in short. (05)

1. What is a sole proprietorship?
A sole proprietorship is a type of business entity where an individual owns and runs the business by themselves. There is no legal distinction between the owner and the business.

2. Definition partnership.
A partnership is a business entity where two or more individuals share ownership and control of the business. The partners share the profits and losses of the business.

3. Describe the business file.
A business file is a collection of documents or records that are used to manage and operate a business. This can include financial records, customer information, employee records, and other important documents.

4. What is data storage file?
A data storage file is a file or collection of files that are used to store and manage data. This can include databases, spreadsheets, text files, and other types of digital files.

5. What is direct access file organization?
Direct access file organization is a method of organizing and accessing data in a file where each record or piece of data is stored in a fixed location. This allows for quick and efficient access to specific records or pieces of data.

Q.2 Answer the followings. (3 Marks Questions.) (Any Five) (15)

Question 1:

What is direct access file organization? Describe its advantages and disadvantages.

Direct access file organization is a file organization technique where each record in the file is stored at a fixed location, determined by a unique key. The key is used to directly access the record in the file.

Advantages:
- Fast retrieval of records: Since the records are stored at fixed locations, direct access files provide fast retrieval of records. - Efficient use of storage space: Direct access files do not require any additional storage space for pointers or indexes.

Disadvantages:
- Limited flexibility: Direct access files do not support efficient addition or deletion of records. Modification of records is also limited. - Limited indexing: Direct access files do not support indexing, which can be useful for searching or sorting records.
